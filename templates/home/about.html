Who am I?

Experience

I feel it’s important that you know that I know what I’m doing. I have a deep working knowledge of the various tools I use which allows me to create awesome websites. Below are my web development tools.

1. CSS

(Cascading Style Sheets)This is a styling language used to style the websites so that they my look attractive.

2. HTML

(Hypertext Markup Language)This is a markup language for the website. It is the skeleton of the website. It is used in structuring the content of the website.

3. Javascript

Javascript is a scripting language. It adds functionality and interactivity to the webpages. Some cool features in websites like sliders and dropdown menus are powered by javascript and some of its awesome libraries.

4. Python

This is a server side programming language. It makes a website more dynamic. It is used by some of the big names like youtube,instagram and netflix.

5. PostgreSQL

This is one of the best and most used database by most websites. Its vast community support makes it reliable and easier to maintain.

5. Linux server administration

Linux is the operating system that powers the majority of the web. I work in the Command line to fine tune and monitor what's taking place on the server.


In conclusion, as both a full time and full-stack web developer i use my skills to help businesses achieve their online goals. I am looking forward to work with you on your project.
